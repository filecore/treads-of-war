Help & FAQ
Controls Simple (WASD): W/S forward/back, A/D turn, Q/E turret, Space/F fire, Tab ammo, V sight, G smoke, C artillery, P/Esc pause. Advanced (original): H/N left track fwd/rev, K/M right track fwd/rev.
Arcade Solo mode — survive endless waves. Each 4 kills upgrades your tank class. You have 3 lives. Heavy class (wave 4+) adds one more enemy per wave.
Attrition You start with a fixed fleet of 5 tanks — losses are permanent. Enemies escalate each battle. Tab switches your controlled tank. Last fleet standing wins.
Strategy Before each battle, spend your budget on any mix of Allied, Soviet and Axis tanks. Enemies buy too. Capture and hold the objective ring for 60 seconds to win the battle. 7 battles total.
LAN Duel 1 v 1 tank battle against a friend — on your local network or over the internet. Requires a relay server running on the host server (see Server Admin below). Enable LAN Duel in Settings, then select it from the mode list. Nothing to install on player machines.
LAN — Playing (Host) Pick your tank on the main menu, then select LAN Duel and click Host Game. A 4-character room code appears (e.g. 7K3M) — share it with your opponent by any means (chat, voice, text). Wait for them to join; the game starts automatically once they connect.
LAN — Playing (Join) Pick your tank, select LAN Duel, then click Find games to see open rooms on the server — click one to fill in the code. Or type the code your opponent gave you directly and click Join. Multiple games can run concurrently; room codes keep them separate.
LAN — Notes Ping is shown in the bottom-left HUD (Advanced Info on). Shot effects — muzzle flash, explosions, ricochets — are synchronised to both screens. Your opponent's tank name and HP bar appear above their vehicle. A Rematch button is shown at the end.
Server Admin — Relay LAN Duel requires a small WebSocket relay service (relay/relay-server.js, Node.js + ws). Add it to your games docker-compose.yaml alongside the nginx service:

relay:
  build: ./relay
  ports:
    - "8765:8765"
  restart: unless-stopped

Then run docker compose up -d --build relay once. The deploy script keeps it updated on each deploy.
Server Admin — NPM Proxy To route the relay through your reverse proxy (no extra port exposed) add a WebSocket location to the Advanced config of your games proxy host in NPM:

location /relay {
  proxy_pass http://192.168.0.101:8765;
  proxy_http_version 1.1;
  proxy_set_header Upgrade $http_upgrade;
  proxy_set_header Connection "upgrade";
  proxy_read_timeout 3600;
}

Replace 192.168.0.101 with your server's LAN IP. All relay traffic then flows through ports 80/443 — no additional firewall rules needed. Works with both ws:// (HTTP) and wss:// (HTTPS) automatically.
Armour Shells are resolved against front / side / rear armour zones. Approach angle determines the zone. A shot that cannot penetrate (firepower < armour × 0.6) ricochets with a gold spark. Aim for flanks and rears of heavy tanks.
Damage Below 50% HP: half speed. Below 25% HP: quarter speed, damage smoke. Below 12% HP: tracks wrecked — hull immobile but turret still operates.
